## ğŸš¦ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository
```bash
git clone https://github.com/shivamkr1563/discord-clone.git
```

2. Navigate to the project directory
```bash
cd discord-clone
```

3. Install dependencies
```bash
npm install
```

4. Run the development server
```bash
npm run dev
```

5. Open your browser and visit `http://localhost:3000` (or the port shown in terminal)

## ğŸ“¦ Build for Production
